# ApexPlanet Task 2 - Blog CRUD Application

## Project Overview

This project is a Blog CRUD Application developed using PHP and MySQL. It allows users to register, log in, and manage blog posts through Create, Read, Update, and Delete operations.

## Features

* User Registration
* User Login
* Password Hashing
* Session Management
* Create Posts
* View Posts
* Edit Posts
* Delete Posts
* Logout Functionality

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* XAMPP

## Database

Database Name: **blog**

### Tables

#### users

* id
* username
* password

#### posts

* id
* title
* content
* created_at
